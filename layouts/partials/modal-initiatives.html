Sistema de automatizacion migratorio para el aeropuerto internacional Arturo Merino Benítez de Santiago

Sistema de automatizacion migratorio para el aeropuerto internacional Arturo Merino Benítez de Santiago que cuenta con un software de reconocimiento biométrico.

Es un sistema de vigilancia constante, permanente y desproporcionado, que no solo almacenaría las imágenes de todos los usuarios que transitan por el aeropuerto, sino que analizaría sus rasgos faciales para compararlos con una base de datos cuyos criterios de elaboración son poco o nada claros.
